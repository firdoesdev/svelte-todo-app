# TODO Apps Using Svelte

# Description
Create An TODO App using Svelte. The App should have the following features:
- Responsive UI
- Implement 3 CRUD
- Implement Dynamic form
- Implement Form Validation using zod
- Implement API
- Implement tanstack query for state management [tanstack](https://tanstack.com/query/v5)
- Implement Social Auth (github and google), using [better-auth](https://github.com/better-auth/better-auth)
- use Drizzle for database
- use Postgres for database
- use Tailwind CSS for styling
- use shadcn for components
- use tailwindcss-sonner for notifications
